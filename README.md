# AICareerBuddy – Backend API

## Opis
Backend API za aplikaciju AICareerBuddy. Omogućuje rad s korisnicima,
AI funkcionalnosti i komunikaciju s mobilnom aplikacijom.

## Tehnologije
- C#
- ASP.NET Core
- Swagger (OpenAPI)
- SQL baza podataka

## Pokretanje
1. Otvoriti projekt u Visual Studio
2. Pokrenuti backend projekt
3. Swagger se automatski otvara u pregledniku

## Swagger
Swagger UI je dostupan na:
`/swagger`

Preko Swaggera je moguće testirati sve API endpointove.

## Deployment
API je deployan i dostupan online.
